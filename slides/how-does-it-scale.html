How does it scale?

  • Databases -> Shard Spaces -> Shards
  • Storage options: LevelDB RocksDB, HyperLevelDBm, LMDB
  • Each shard is its own database
  • Log Structured Merge Tree
  • B+ Tree
  • Shards can be replicated to multiple servers
  • Multiple shards for a period of time, prevents write hot spots
